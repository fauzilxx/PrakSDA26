# Tugas 2: Implementasi Struktur Data Dasar dengan Java

## Deskripsi Tugas

Pada praktikum ini, praktikan diminta untuk mengimplementasikan struktur data dasar secara manual menggunakan bahasa Java tanpa menggunakan `java.util`. Tujuan tugas ini adalah agar praktikan memahami cara kerja struktur data dari proses dasar pembentukannya, bukan hanya menggunakannya dari library bawaan Java.

## Pilihan Struktur Data

Praktikan diminta untuk memilih **minimal 1** struktur data dari daftar berikut:

1. ArrayList
2. LinkedList
3. Stack
4. Queue

### Nilai Tambahan

- Jika praktikan mengerjakan **1 struktur data**, maka dinilai sebagai tugas utama.
- Jika praktikan mengerjakan **2 struktur data atau lebih**, maka akan mendapat **nilai plus**.

## Ketentuan Pengerjaan

- **Tidak boleh menggunakan `java.util`** untuk implementasi struktur data yang dipilih.
- Program harus dibuat sendiri secara manual menggunakan `class`, `atribut`, dan `method`.
- Setiap kode program wajib diberi komentar yang jelas agar alur program mudah dipahami.
- Program harus dapat di-compile dan di-run dengan baik.
- Praktikan bebas menggunakan data bertipe `int`, `String`, atau tipe data sederhana lainnya.

## Fitur Minimal Program

### 1. Jika memilih ArrayList

Program minimal memiliki fitur:
- Menambahkan data
- Menampilkan seluruh data
- Mengambil data berdasarkan index
- Mengubah data pada index tertentu
- Menghapus data
- Menampilkan ukuran array list

### 2. Jika memilih LinkedList

Program minimal memiliki fitur:
- Menambahkan node
- Menampilkan isi linked list
- Menghapus node
- Mencari data
- Menambahkan data di awal atau akhir

> **Catatan:** Jika membuat Doubly Linked List dan dapat melakukan traversal dua arah, akan mendapat nilai tambahan.

### 3. Jika memilih Stack

Program minimal memiliki fitur:
- `push`
- `pop`
- `peek`
- `isEmpty`
- Menampilkan isi stack

### 4. Jika memilih Queue

Program minimal memiliki fitur:
- `enqueue`
- `dequeue`
- `peek` / `front`
- `isEmpty`
- Menampilkan isi queue

## Ketentuan Pengumpulan

- File yang dikumpulkan adalah source code `.java`
- Tidak perlu membuat laporan
- Kode wajib memiliki komentar yang jelas
- Jika mengerjakan lebih dari 1 struktur data, maka semua file `.java` dikumpulkan (boleh dalam bentuk zip)
- Format nama file: `KELAS_SDA_NO-PRAKTIKUM_NIM_NAMA`
