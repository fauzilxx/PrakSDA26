# Penugasan Responsi 1

## 1. Lampiran Template Laporan

Gunakan template laporan berikut sebagai acuan penulisan:

[Template Laporan Responsi](https://docs.google.com/document/d/1gSQCFZBnX8pdXQRq1YgEZYCz0hAIiOmFSxRAgApRAt8/edit?usp=sharing)

## 2. Deskripsi Tugas

Buatlah sebuah program menggunakan bahasa **Java** dengan tema bebas.

Program wajib menerapkan struktur data dan disertai analisis kompleksitas dengan rincian poin sebagai berikut:

- List: **5 poin**
- Stack: **5 poin**
- Queue: **5 poin**
- Set: **5 poin**
- Map: **5 poin**
- Tree: **15 poin**
- Binary Tree: **10 poin**

**Total keseluruhan poin: 50 poin**

## 3. Catatan Penilaian

Poin dihitung berdasarkan **jenis struktur data**, bukan banyaknya struktur data.

Contoh:

- Jika dalam program terdapat `List` sebanyak 3 buah, maka nilai untuk kategori List tetap **5 poin**, bukan 15 poin.

## 4. Penilaian Laporan Responsi

- Nilai kelengkapan laporan: **0-20**
- Nilai kerapian: **0-10**
- Nilai analisis kompleksitas: **0-20**

**Total Kode + Laporan = 100**

## 5. Ketentuan Pengumpulan

Kumpulkan dalam bentuk **1 file `.zip`** yang berisi:

1. File source code Java (`*.java`)
2. File laporan dalam bentuk PDF (`*.pdf`)

## 6. Format Nama File

Gunakan format nama file berikut:

`NIM_Nama_Responsi1.zip`

Contoh:

`L0123456_AhmadSigmew_Responsi1.zip`
